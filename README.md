### 1. Merge Unique Elements

Write a Java program to merge two ArrayLists of **books** and remove duplicates.

### 2. Find Maximum Element

Write a Java program to find the **longest string** (by length) in an ArrayList of **movie titles**.

### 3. Count Occurrences

Write a Java program to count how many times **"Pizza"** appears in an ArrayList of **foods**.

### 4. Remove All Matching Elements

Write a Java program to remove all **"Banana"** entries from an ArrayList of **fruits**.

### 5. Replace All Elements

Write a Java program to replace every element in an ArrayList of **colors** with **"Black"**.

### 6. Check Sublist Presence

Write a Java program to check if an ArrayList of **sports** contains another smaller ArrayList
(e.g., `["Football", "Tennis"]`).

### 7. Find Index of Element

Write a Java program to find the **first and last index** of **"Rose"** in an ArrayList of **flowers**.

### 8. Remove Duplicates

Write a Java program to remove duplicate elements from an ArrayList of **animals**.

### 9. Convert ArrayList to Array

Write a Java program to convert an ArrayList of **cities** into a regular array.

### 10. Convert Array to ArrayList

Write a Java program to convert an array of **numbers** into an ArrayList.

---

### 11. Find Common Elements

Write a Java program to find common elements between two ArrayLists of **countries**.

### 12. Remove Even-Length Strings

Write a Java program to remove all strings with **even length** from an ArrayList of **names**.

### 13. Find Smallest Element

Write a Java program to find the **shortest string** in an ArrayList of **songs**.

### 14. Replace All Vowels

Write a Java program to replace all **vowels** in each string of an ArrayList of **words** with `*`.

### 15. Partition ArrayList

Write a Java program to split an ArrayList of **numbers** into two lists: **even** and **odd**.

### 16. Rotate Elements

Write a Java program to rotate elements of an ArrayList of **days of the week** by **2 positions**.

### 17. Remove Null Values

Write a Java program to remove all **null values** from an ArrayList of **student names**.

### 18. Find Second Largest Element

Write a Java program to find the **second longest string** (by length) in an ArrayList of **movie titles**.

### 19. Replace Elements with Lengths

Write a Java program to replace each string in an ArrayList of **fruits** with its **length**.

### 20. Create Nested ArrayList

Write a Java program to create an ArrayList of ArrayLists
(e.g., **departments with employee names**) and print them.
